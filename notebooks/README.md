# Notebooks

Interactive Jupyter notebooks for AISE26 W20D4.

## Notebooks Included

- `w20d4_scientific_playground.ipynb` - Main interactive notebook

## Usage

1. Make sure you have Jupyter installed:
   ```bash
   pip install jupyter notebook
   ```

2. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

3. Open the playground notebook and run cells step-by-step.

## Tips

- Run cells in order for best results
- Save your work frequently
- Check outputs/ directory for generated files
